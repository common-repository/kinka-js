== Description ==

8 bit cute birds (Zebra finches) fly around your website.

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “kinka.js”.
3. Select the “kinka.js” plugin and click “Install”.
4. Activate the plugin.

5. `<div class="kinka">` Please insert this tag in the web page. (As many birds as this tag are displayed.)
    * Pages
    * [Appearance] -> [Editor] PHP files
    * Posts
3. Add `perch` to the class of HTML tag that already exists. ex) `<div class="page-header perch"></div>`
    * Bird sit on the tag which added the perch class.
